# API Pública EduJarvis (v1)

## Autenticação
As requisições devem incluir o Header `X-API-Key`.

## Endpoints Principais

### POST `/api/public/v1/chat`
Envia uma mensagem para o orquestrador inteligente.

### GET `/api/public/v1/reports`
Recupera relatórios narrativos gerados por IA.

### POST `/api/public/v1/assessments`
Solicita a correção automática de uma atividade.

---
*Para chaves de API, acesse o Enterprise Command Center.*
